# Восстановление матрицы

**Задача:** Есть матрица. Квадратная. Элементы матрицы целые числа. Матрицу представили как набор строк и столбцов. Получили 2N векторов каждый длиной N. порядок и разметка – кто строка кто столбец - потеряны. Нужно восстановить исходную матрицу (с точностью то транспозиций).

**Ассимптотика**: O(n!)
